propuesto y al video que vieron, identifiquen cada fase de un DBLC en el sistema supuesto
y detallen cuáles serían las actividades a desarrollar en cada una.

El analisis (Estudio inicial de una base de datos)
* Cuestionar al usuario sobre el sistema
* Averiguar que caracteristsicas y restricciones extra debe tener el sistema (segun el usuario)
* Se debe armar una especificacíon funcional y un análisis estructurado con requerimientos
* El usuario debe revisar que todo este bien en estos documentos
* Se pueden desarrolar prototipos como para analizar como seria el sistema


El diseño (Diseño de una base de datos)
* Definir definitivamente la estructura y forma definitiva de un sistema
* Armar graficos y diagramas de estructura.
* Hacer una especificación detallada con toda la documentacion del programa planificado
* Se define el diseño externo y de datos


El desarrollo (Implementacion y carga, operacion)
* Se desarrolla el software
  * Se diseña
  * Se codifica
  * Se depura
  * Alternativamente se puede comprar software existente o usar software de codigo abierto si existe alguno que ya cumple la funcion requereida suficientemente bien 
* Se adquiere el hardware (en el caso que lo ya existente no vaste)
* Se dearrolla y suministra la documentación


La prueba (Prueba y evaluación)
* Se usan datos de muestra para probar el sistema
* Los usuarios pruban el sistema y hacen observaciónes


El mantenimiento y la Evolución (Mantenimiento y evolucion)
* Se hace el mantenimiento correspondiente al sistema
